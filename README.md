# Дневник погоды (Weather Diary)

**Автор:** Чемезова Дарья Александровна

## Описание программы

"Дневник погоды" — это графическое приложение для ведения записей о погоде.  
Позволяет добавлять, просматривать, фильтровать, сохранять и загружать данные о погоде в формате JSON.

## Как запустить приложение

1. Убедитесь, что установлен Python 3.6 или новее.
2. Сохраните файл `weather_diary.py` в любую папку.
3. Запустите скрипт командой:

```bash
python weather_diary.py
